# Bank Management System (Console Based)

A simple Java console-based Bank Management System built using Core Java concepts.

## 🚀 Features

- Create Bank Account
- Deposit Money
- Withdraw Money
- Check Balance
- View Account Details
- Menu-driven interface

## 🧠 Concepts Used

- Class & Object
- Encapsulation
- Constructors
- Static & Non-Static Members
- Arrays (future upgrade)
- Method Overloading
- Control Statements (if-else, switch, loops)
- Scanner for dynamic input

## 🏗 Project Structure

BankManagementSystem  
│  
├── src/com/bank/app  
│   ├── BankAccount.java  
│   ├── Main.java  

## ▶️ How to Run

1. Clone the repository
2. Open in Eclipse IDE
3. Run Main.java
